# 🔥 Forest Fire Prediction System

## 📌 Overview

This is my **first Machine Learning project**, where I built a predictive system to estimate forest fire risk based on environmental conditions.

The project is based on the **Algerian Forest Fire Dataset** and uses a **Ridge Regression model**. It is deployed as a web application using Flask.

* User-friendly web interface
* Real-time prediction system

* ML model integration (Ridge Regression)

## ⚙️ How to Run Locally

### 1️⃣ Clone the repository

```
git clone https://github.com/LAKSHYACH543/testforestfire.git
```

### 2️⃣ Navigate to project folder

```
cd testforestfire
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the application

```
python application.py
```

### 5️⃣ Open in browser

```
http://127.0.0.1:5000/
```

## 📈 Model Information

* Dataset: **Algerian Forest Fire Dataset**
* Algorithm: Ridge Regression
* Features used:

  * Temperature
  * RH (Relative Humidity)
  * Wind Speed (Ws)
  * Rain
  * FFMC
  * DMC
  * ISI
  * Classes
  * Region
